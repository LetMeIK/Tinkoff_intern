            пассажиров больше 30.


`Задача 3.` Выведите ТОП 10 пилотов-капитанов (first_pilot_id), которые совершили наибольшее
            число грузовых перелетов в этом году.
            
            
Файл SQL_ALLIN.sql Содержит все 3 задачи всместе
